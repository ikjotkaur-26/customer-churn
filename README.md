# Customer Churn Analysis

This project analyzes customer churn using a European bank dataset. I used SQL to explore the data and calculate business KPIs, and Power BI to build an interactive dashboard for visualizing customer behavior and churn patterns.

## Tools Used

- SQL (MySQL)
- Power BI
- Microsoft Excel (Dataset)

## SQL Analysis

The following SQL queries were used to analyze the dataset:

- Total Customers
- Active Customers
- Churned Customers
- Churn Rate
- Average Balance
- Average Salary
- Customers by Geography
- Churn Rate by Geography
- Top 5 Customers by Balance

## Dashboard Features

- KPI Cards
- Customer Churn Overview
- Age Distribution
- Average Balance by Number of Products
- Balance vs Estimated Salary
- Customer Insights

## Key Insights

- Around 20% of customers have churned.
- Germany has the highest churn rate.
- Inactive customers are more likely to churn.
- Most customers belong to the 30–45 age group.
- High-value customers can be identified based on account balance.
